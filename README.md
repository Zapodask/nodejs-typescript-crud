# Crud com nodejs typescript

## Tecnologias utilizadas

* Nodejs
* TypeScript
* Express

## Setup

1. Instale os pacotes
    ```bash
    yarn

    # Ou

    npm install
    ```

2. Inicie o projeto
    ```bash
    yarn dev

    # Ou

    npm run dev
    ```
